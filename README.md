# Корпус 
Я взяла корпус анекдотов из kaggle. Почему именно его? Потому что смотреть на данные анкедотов весело и полезно (можно выучить парочку новых). Для эксперимента была сформирована выборка из 1500 текстов.

PS. Позже я поняла, что это сборник самых НЕсмешных шуток. Они ввергают в полное уныние.


# Модули проекта

## `preprocessing.py` (предобработка текста)

- `preprocess_text()` (одного текста)
  - приведение текста к нижнему регистру;
  - удаление пунктуации и цифр;
  - токенизация ("razdel");
  - удаление стоп-слов; 
  - лемматизация ("pymorphy3");
  - удаление дубликатов и пустых документов

- `preprocess_corpus()`  
  предобработка всего корпуса

## `dict_indexes.py`

Модуль реализует **обратный индекс через словари**.

- построение частотного индекса
- расчёт статистик для BM25
- поиск через частотный индекс
- поиск через BM25

Основная структура индекса: 
> term → {doc_id: tf}

## `matrix_indexes.py`

Модуль реализует **матричную версию индексов**.

Используются библиотеки:
- `numpy`
- `scipy`

Реализовано:

- построение матрицы `document × term`
- матричный частотный поиск
- матричный BM25

Поиск выполняется через матричное умножение:
> scores = matrix @ query_vector

## `main.py`

Этот модуль:

1. загружает корпус
2. выполняет предобработку
3. строит индексы
4. принимает пользовательский запрос
5. возвращает топ релевантных документов

# Запуск
Сначала установите зависимости:

```bash
pip install -r requirements.txt
```
Пример:
```
python main.py \
  --input data/russian_jokes_1500.csv \
  --text-col text \
  --query "муж жена" \
  --backend dict_bm25 \
  --top-k 5
```
Параметр --backend определяет используемый индекс
```
dict_freq
dict_bm25
matrix_freq
matrix_bm25
```
